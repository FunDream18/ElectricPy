reactance¶

electricpy.reactance(z, f=60, sensetivity=1e-12)¶

reactance Function:

Calculates the Capacitance or Inductance in Farads or Henreys (respectively) provided the impedance of an element. Will return capacitance (in Farads) if ohmic impedance is negative, or inductance (in Henrys) if ohmic impedance is positive. If imaginary: calculate with j factor (imaginary number).

Parameters

  • z (complex) – The Impedance Provided, may be complex (R+jI)

  • f (float, optional) – The Frequency Base for Provided Impedance, default=60

  • sensetivity (float, optional) – The sensetivity used to check if a resistance was provided, default=1e-12

Returns

out – Capacitance or Inductance of Impedance

Return type

float
