================
FAULT Sub-Module
================

Loaded with functions and assistants designed for fault calculations and
electrical system protection design, the `electricpy.fault.py` submodule is just
the tool electrical engineers need to aid in quick analysis and design.

Importing is reccomended as a command similar to the following command.

>>> import electricpy.fault as flt
