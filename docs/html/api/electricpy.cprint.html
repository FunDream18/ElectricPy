cprint¶

electricpy.cprint(val, unit='', label='', printval=True, ret=False, round=3)¶

cprint Function

This function is designed to accept a complex value (val) and print the value in the standard electrical engineering notation:

magnitude ∠ angle °

This function will print the magnitude in degrees, and can print a unit and label in addition to the value itself.

Parameters

  • val (complex) – The Complex Value to be Printed, may be singular value, tuple of values, or list/array.

  • unit (string, optional) – The string to be printed corresponding to the unit mark. default=””

  • label (string, optional) – The pre-pended string used as a descriptive labeling string. default=””

  • printval (bool, optional) – Control argument enabling/disabling printing of the string. default=True

  • ret (bool, optional) – Control argument allowing the evaluated value to be returned. default=False

  • round (int, optional) – Control argument specifying how many decimals of the complex value to be printed. May be negative to round to spaces to the left of the decimal place (follows standard round() functionality). default=3

Returns

numarr – The array of values corresponding to the magnitude and angle, values are returned in the form: [[ mag, ang ],…,[ mag, ang ]] where the angles are evaluated in degrees.

Return type

numpy.ndarray
