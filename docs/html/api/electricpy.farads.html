farads¶

electricpy.farads(VAR, V, freq=60)¶

Capacitance from Apparent Power Formula

Function to calculate the required capacitance in Farads to provide the desired power rating (VARs).

\[C = \frac{VAR}{2*\pi*freq*V^2}\]
Parameters

  • VAR (float) – The rated power to meet.

  • V (float) – The voltage across the capacitor; not described as VLL or VLN, merely the capacitor voltage.

  • freq (float, optional) – The System frequency

Returns

C – The evaluated capacitance (in Farads).

Return type

float
