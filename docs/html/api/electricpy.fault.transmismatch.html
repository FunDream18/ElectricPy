transmismatch¶

electricpy.fault.transmismatch(I1, I2, tap1, tap2)¶

Transformer TAP Mismatch Function

Function to evaluate the transformer ratio mismatch for protection.

Parameters

  • I1 (complex) – Current (in amps) on transformer primary side.

  • I2 (complex) – Current (in amps) on transformer secondary.

  • tap1 (float) – Relay TAP setting on the primary side.

  • tap2 (float) – Relay TAP setting on the secondary side.

Returns

mismatch – Transformer CT mismatch value associated with relay.

Return type

float
