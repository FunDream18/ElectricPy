fftplot¶

electricpy.fftplot(dc, real, imag=None, title='Fourier Coefficients')¶

FFT System Plotter

Plotting function for FFT (harmonic) values, plots the DC, Real, and Imaginary components.

Parameters

  • dc (float) – The DC offset term

  • real (list of float) – Real terms of FFT (cosine terms)

  • imag (list of float, optional) – Imaginary terms of FFT (sine terms)

  • title (str, optional) – String appended to plot title, default=”Fourier Coefficients”
