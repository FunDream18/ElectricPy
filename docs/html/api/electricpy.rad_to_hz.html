rad_to_hz¶

electricpy.rad_to_hz(radians)¶

Simple Radians to Hertz Converter

Accepts a frequency in radians/sec and calculates the hertzian frequency (in Hz).

\[f_{\text{Hz}} = \frac{f_{\text{rad/sec}}}{2\cdot\pi}\]

Same as hertz.
