sequencez¶

electricpy.sequencez(Zabc, reference='A', round=3)¶

Sequence Impedance Calculator

Accepts the phase (ABC-domain) impedances for a system and calculates the sequence (012-domain) impedances for the same system.

Parameters

  • Zabc (numpy.ndarray of complex) – 2-D (3x3) matrix of complex values representing the phasor impedances in the ABC-domain.

  • reference ({'A', 'B', 'C'}) – Single character denoting the reference, default=’A’

  • round (int, optional) – Integer denoting number of decimal places resulting matrix should be rounded to. default=3

Returns

Z012 – 2-D (3x3) matrix of complex values representing the sequence impedances in the 012-domain

Return type

numpy.ndarray of complex
