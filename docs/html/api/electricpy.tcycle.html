tcycle¶

electricpy.tcycle(ncycles=1, freq=60)¶

Time of Electrical Cycles

Evaluates the time for a number of n cycles given the system frequency.

\[t = \frac{n_{cycles}}{freq}\]
Parameters

  • ncycles (float, optional) – Number (n) of cycles to evaluate, default=1

  • freq (float, optional) – System frequency in Hz, default=60

Returns

t – Total time for ncycles

Return type

float
