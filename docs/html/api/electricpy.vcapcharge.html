vcapcharge¶

electricpy.vcapcharge(t, Vs, R, C)¶

Charging Capacitor Voltage

Function to calculate the voltage of a capacitor that is charging given the time.

\[V_c=V_s*(1-e^{\frac{-t}{R*C}})\]
Parameters

  • t (float) – The time at which to calculate the voltage.

  • Vs (float) – The charging voltage for the capacitor.

  • R (float) – The ohmic value of the resistor being used to discharge.

  • C (float) – Capacitive value (in Farads).

Returns

Vc – The calculated voltage of the capacitor.

Return type

float
