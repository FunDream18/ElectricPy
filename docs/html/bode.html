BODE Sub-Module¶

Specifically designed to support Bode-Plotting function routines, this submodule supports several methods of generating Bode plots showing both magnitude and angle.

It is reccomended that the electricpy.bode.py submodule be imported using a command similar to the one below.

>>> import electricpy.bode as bode
