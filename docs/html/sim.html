SIM Sub-Module¶

Loaded with functions and assistants designed for system modeling and simulation applicable for electrical system design, the electricpy.sim.py submodule is just the tool electrical engineers need to aid in quick analysis and design.

Importing is reccomended as a command similar to the following command.

>>> import electricpy.sim as sim
